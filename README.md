# Ajo's Blog

Personal blog for cloud, DevOps, automation, and security notes.

This repository has been converted from Jekyll to Hugo and uses the [m10c theme](https://themes.gohugo.io/themes/hugo-theme-m10c/).

## Local development

```bash
hugo server
```

The site is served at `http://localhost:1313/`.

## Publishing

GitHub Pages publishing is handled by `.github/workflows/hugo.yml`.
